Website

  • 문구랜드

    쇼핑몰「문구랜드」

    기존 사이트의 난잡한 내용 구성과,
    문구류 쇼핑몰의 느낌을 좀 더 살리기 위해
    이 사이트를 선정하여, 메인을 리뉴얼해보았습니다.

    사용툴 : Adobe Photoshop,EditPlus

  • 라인

    기업사이트「LINE」

    해당 사이트의 단순한 구성에도 탄탄한 내용을 전달함에
    인상을 받고, 선정하여 카피해보았습니다.

    사용툴 : Adobe Photoshop,Brackets

  • 삼표

    기업사이트「삼표」

    기존 사이트의 Footer 내의 메뉴바의 불편함을 개선하고자,
    부분 리뉴얼방식으로 카피사이트를 새롭게 제작해보았습니다.

    사용툴 : Adobe Photoshop,Visual Studio Code

  • 카카오

    기업사이트「KAKAO」

    '카카오'라하면 떠오르는 이미지와는 이질적인 느낌이 들어서,
    부분적으로 컨텐츠를 추가하여, 리뉴얼해보았습니다.

    사용툴 : Adobe Photoshop,Brackets

    상업적 리뉴얼이 아닌 개인 리뉴얼작업입니다.
    공식 사이트와는 관련없습니다.

  • 원페이지

    원페이지「FOOLS GARDEN」

    관심분야를 바탕으로 원페이지를 제작함으로써,
    한 페이지에 어떻게 정보전달을 알차게할 수 있을지,
    고민하며 제작해보았습니다.

    사용툴 : Adobe Photoshop,Brackets
